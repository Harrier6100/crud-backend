概要

計量実績を取得・検索するためのWebAPIです。

エンドポイント

メソッド パス 説明
GET /keiryoujisseki 計量実績を取得する。条件指定による検索が可能。

GET /keiryoujisseki

説明

計量実績を取得する。条件指定による検索が可能。

パラメータ

名前 必須 説明
norilotno 任意 計画IDを完全一致で検索する。
koujoulotno 任意 工場ロット番号を完全一致で検索する。
field 任意 レスポンスに含める項目を指定する。複数指定する場合はカンマ区切りとする。未指定の場合は全項目を含める。
sort 任意 ソート条件を指定する。形式：<項目名> [asc|desc] asc は省略可（省略時は昇順）。複数指定する場合はカンマ区切りとする。

レスポンス

名前 型 説明
tokoukino string 塗工機番号
norilotno string 製糊ロット番号
seizoudate string 製造日
noriname string 粘着剤名
seikokaisi string 製糊開始
seikoshuryou string 製糊終了
kaisidate string 撹拌開始時間
shuryoudate string 撹拌終了時間
ondo number 温度
ondogouhi string 温度判定 0：未検査 1：合格 2：不合格 3：検査不要
nendo number 粘度
nendogouhi string 粘度判定 0：未検査 1：合格 2：不合格 3：検査不要
aniline string アニリン検査 0：未検査 1：合格 2：不合格 3：検査不要
koutoucd string 工棟コード
kishakugenzairyoucd string 希釈溶剤原材料コード
kishakugenzairyouname string 希釈溶剤原材料名
kishakucount number 希釈溶剤使用量
norigouhi string 製糊合否 0：未検査 1：合格 2：不合格 3：検査不要
uchikiriflag string 打ち切りフラグ 1：打ち切り NULL：有効
uchikiridate string 打ち切り日時
kamakikicd string 撹拌釜機材コード
kamakikiname string 攪拌釜機材名
grindgauge string グラインドゲージ検査 0：未検査 1：合格 2：不合格 3：検査不要
recount number 再実行回数、製糊作業中にバッチ再計算を行った回数
hojogamaflag string 補助釜フラグ 1：補助釜の実績 1以外：製糊の実績
hojogamatounyuflag string 補助釜投入フラグ 1：製糊釜へ投入済 1以外：未投入 ※補助釜フラグ1の場合のみ使用
maintetime string 更新日時
kakuhanrpm number 撹拌機回転数
shokubaitounyudate string 触媒投入日時
sireino string 指令番号、製糊ロット番号に紐づく塗工指令番号
genzairyoucd string 原材料コード
genzairyouname string 原材料名
sijicount number 計量指示値
count number 計量実績値
chouseicount number 調整量、ゼロクリアのために調整した量
zancount number 残量、計量実績値を引落した後の在庫残量
zeroclearflag string ゼロクリアフラグ、残量をゼロにした場合に1を立てる。それ以外は0
koujoulotno string 工場ロット番号
makerlotno string 仕入先ロット番号
tantoucd string 担当者コード
tantouname string 担当者名
handyno string ハンディ番号
keiryoukekka string 計量結果 0：未判定 1：計量OK 2：計量NG
kowakecardno string 小分けカード番号
kowakekikicd string 小分け容器機材コード、使用した小分け容器の機材コードを記録する。材料容器計量、主剤の計量・投入時はNULL
kowakekikiname string 機器名、使用した小分け容器の機材名
tounyuflag string 投入フラグ 1：投入済 1以外：未投入
bikou string 備考
kowakesikibetuid string 小分け容器識別ID
keiryoudate string 計量日時、計量値を記録した日時
kikicd string 使用機器コード、計量に使用した機器コードを記録する。手入力の場合はNULL
kikiname string 機器名、計量に使用した機器名
shoridate string 処理日時、計量実績のレコード更新日時
